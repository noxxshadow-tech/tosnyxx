🔒 En bref : Nyxx ne collecte aucune donnée personnelle sur les utilisateurs. Seules les configurations des serveurs Discord sont enregistrées pour permettre le bon fonctionnement du bot.

01 / COLLECTE

Données collectées

Nyxx collecte uniquement les données strictement nécessaires à son fonctionnement :

  • Configurations et paramètres des serveurs Discord
  • Préférences d'administration définies par les gestionnaires du serveur
PAS de messages PAS de données personnelles PAS d'IDs utilisateurs
02 / UTILISATION

Utilisation des données

Les configurations collectées sont utilisées exclusivement pour :

  • Mémoriser les paramètres du bot sur votre serveur
  • Assurer un fonctionnement cohérent entre les sessions
03 / STOCKAGE

Stockage et sécurité

Les données de configuration sont stockées de manière sécurisée. Elles ne sont jamais partagées, vendues ou transmises à des tiers. L'accès est limité aux seuls développeurs du bot.

04 / SUPPRESSION

Suppression des données

Lorsque Nyxx est retiré d'un serveur, les configurations associées à ce serveur peuvent être supprimées sur demande. Pour toute demande de suppression, contactez les développeurs via Discord.

05 / TIERS

Services tiers

Nyxx opère sur la plateforme Discord. Son utilisation est soumise à la Politique de Confidentialité de Discord. Aucun autre service tiers n'est impliqué dans le traitement des données.

06 / CONTACT

Nous contacter

Pour toute question relative à cette politique ou à vos données, vous pouvez nous contacter directement via Discord. Nous nous engageons à répondre dans les meilleurs délais.
